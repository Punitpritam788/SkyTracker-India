# ✈️ SkyTracker India

> **Smart Flight Pricing Intelligence for Indian Domestic Flights**

A sophisticated flight pricing tracker that monitors airfare trends across India's 70+ airports, visualizes 330-day booking lifecycles, forecasts future fares, and empowers travelers to make informed, cost-effective flight decisions.

## 🎯 Key Features

### 📊 **Price Lifecycle Tracking**
- Monitor complete 330-day booking window from flight listing to departure
- Real-time price history with visual tracking
- Understand tier-based dynamic pricing across Indian routes
- Historical data analysis from T-330 to T-0 days

### 📈 **Intelligent Forecasting**
- AI-powered price predictions at departure
- Identify peak and valley periods in booking window
- Forecast confidence breakdown by confidence level
- Trend analysis with 7-day, 30-day, 90-day, and past views

### 💡 **Smart Booking Score (0-100)**
- Dynamic deal rating based on:
  - Current fare position vs. forecast
  - Days remaining to departure
  - Historical price trajectory
- Grade system: A+ (Exceptional) → D (Poor Timing)
- Contextual recommendations for booking decisions

### 📅 **Flex Date Finder**
- Compare prices ±3 days around your preferred departure
- Identify cheapest dates at a glance
- Single-click re-tracking for alternative dates
- Visual price comparison with "BEST" badge

### 🗓️ **Best Day to Fly Analysis**
- Average fare breakdown by weekday (Mon-Sun)
- Identify peak and budget-friendly days
- Seasonal demand patterns for your route
- Weekend vs. weekday pricing insights

### 💰 **Price Alert System**
- Set custom target fares for any route
- Get notified of achievable booking dates
- Multiple sort options: by date or price
- Visual indicators for past, current, and future matches

### 🔥 **Route Demand Indicator**
- Real-time demand levels for tracked routes
- Understand how demand drives fare escalation
- Tier-based analysis (Metro/Capital/Regional hubs)
- Social proof showing travelers watching your route

### 🎨 **Advanced Analytics**
- **Price Heatmap Calendar**: 28-day visual price grid (cheap → expensive)
- **Route Comparison**: Compare your route against alternative destinations
- **Fare History Breakdown**: Seasonal patterns and baseline pricing
- **Live Route Path**: Interactive map showing flight corridor
- **Forecast Confidence**: Detailed breakdown of prediction reliability

## 🚀 Quick Start

### **Online Access**
Simply open `index.html` in any modern web browser. No installation required!

```bash
# Clone the repository
git clone https://github.com/Punitpritam788/SkyTracker-India.git

# Open in browser
cd SkyTracker-India
open index.html  # macOS
# or
start index.html  # Windows
# or
xdg-open index.html  # Linux
```
